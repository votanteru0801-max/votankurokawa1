# 月額費用概算

**確認日: 2026-07-16**（料金は変更されるため、契約前に必ず各社公式ページで最新情報を確認してください。為替レートは目安として1ドル=150円で概算しています）。

利用者が石橋輝一1名のみという極めて小規模な利用を前提に、最低利用・標準利用・多めの利用の3パターンで概算します。

## 前提利用量

| パターン | AI分析リクエスト数/月 | LINEメッセージ数/月 |
|---|---|---|
| 最低利用 | 約30回 | 約100通 |
| 標準利用 | 約150回 | 約400通 |
| 多めの利用 | 約500回 | 約1,200通 |

## 1. Anthropic API（Claude）
確認先: https://platform.claude.com/docs/en/about-claude/pricing （確認日2026-07-16時点、Claude Sonnet系: 標準価格 入力$3/百万トークン・出力$15/百万トークン。2026年8月末までの導入価格は入力$2・出力$10/百万トークンとの情報あり、変動する可能性あり）

1リクエストあたり概算: 入力約3,000トークン（命式データ+人事情報+システムプロンプト）、出力約800トークンと仮定 → 約$0.021/回

| パターン | 月額(USD) | 月額(概算円) |
|---|---|---|
| 最低利用（30回） | 約$0.6 | 約100円 |
| 標準利用（150回） | 約$3.2 | 約470円 |
| 多めの利用（500回） | 約$10.5 | 約1,600円 |

## 2. LINE公式アカウント
確認先: https://developers.line.biz/ja/docs/messaging-api/pricing/ （確認日2026-07-16、フリー: 0円/200通まで、ライト: 5,000円/5,000通まで、スタンダード: 15,000円/30,000通+従量。2026年10月改定予定あり）

石橋輝一1名との会話のみのため、月間メッセージ数はいずれのパターンでもフリープラン(200通)〜ライトプラン(5,000通)の範囲に収まる見込みです。

| パターン | プラン目安 | 月額 |
|---|---|---|
| 最低利用（約100通） | フリープラン | 0円 |
| 標準利用（約400通） | フリープラン | 0円 |
| 多めの利用（約1,200通） | フリープラン〜ライトプラン | 0〜5,000円 |

## 合計概算

| パターン | 月額概算(円) |
|---|---|
| 最低利用 | 約1,600〜1,700円 |
| 標準利用 | 約2,000円 |
| 多めの利用 | 約3,300〜8,300円（LINEプラン次第） |

**この規模の運用では、Cloud SQLの固定費（常時起動費用）が費用の大部分を占めます。** 費用をさらに抑えたい場合は、開発・検証時のみCloud SQLを起動する運用や、より小さいインスタンスクラスの検討を推奨します。
